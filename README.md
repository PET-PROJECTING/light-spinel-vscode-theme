# Light Spinel

A light, refined color theme for Cursor and VS Code.

## Installation

### From Cursor / VS Code Marketplace

1. Open the Extensions view (`Ctrl+Shift+X` or `Cmd+Shift+X`)
2. Search for **Light Spinel**
3. Click **Install**

### From OpenVSX (Cursor)

If the theme is published to OpenVSX, it will appear in Cursor's extension marketplace automatically.

### Manual Install

1. Download the `.vsix` file from the [Releases](https://github.com/PET-PROJECTING/light-spinel-vscode-theme/releases) page
2. In Cursor/VS Code: `Extensions` → `...` → `Install from VSIX...`
3. Select the downloaded file

## Usage

1. Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
2. Run **Preferences: Color Theme**
3. Select **Light Spinel**

Or: `File` → `Preferences` → `Theme` → `Color Theme` → **Light Spinel**
